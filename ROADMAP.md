# Roadmap

## Near-Term

- Improve menu bar UX and discoverability
- Stabilize permissions flow and diagnostics
- Polish About/Help experience and project metadata links
- Keep release pipeline deterministic and reproducible

## Mid-Term

- Expand localization coverage beyond EN/DE
- Add richer automated checks for app launch/menu behavior
- Improve packaging metadata and release notes quality

## Future Ideas

- Optional onboarding tips for first launch
- More configurable paste behavior profiles
- Better troubleshooting export for user bug reports

## Process

Roadmap priorities are updated based on:

- User feedback from issues/discussions
- Stability and regression risk
- Maintenance effort vs. user value
